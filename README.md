# Classroom Subject and Time Tracker (CSTT)
CSTT or the Classroom Subject and Time Tracker aims to aid my current class' awareness regarding the subject and time, solely developed by **Jazzminum** for the **"thing-on-desk" YSWS event** from **Hack Club**.

## Project Functionality Summary
It primarily is designed to have a **built-in timer** to boost class productivity and properly track the time spent during activities such as groupworks, tests, and activities; as well as have a **visual schedule indicator** on the side to show what the current day's schedule is supposed to be. 

## Why Was This Developed?
I've noticed that most of our teachers in class utilize timers whenever we need to accomplish academic-related tasks. I thought of creating our class's own custom timer (based on our class' theme), to have a centralized timer across all subjects. I also decided to add in the visual schedule indicator since most of the people in my class still ask for the upcoming subjects during the day; and I figured it'd be nice to see on the screen whenever we have activities so that we can prepare ourselves for the next class.

## Features
**I. Simple Functional Timer**
- Visual Representation of Time Left
- Ability to Pause, Stop, and Start Timer
- Ability to Change Timer Amount

**II. Schedule Indicator**
- Sidebar Showing Current Day's Schedule
- Ability to Change Depending on Current Day

**III. Planned Features**
*These are features that probably won't be implemented before the end of the "thing-on-desk" YSWS event, but are stuff I would still like to add in the future.*
- Polished Responsive Design (It's kind of scuffed at the moment, but it exists)
- Dashboard showing the total time spent doing schoolworks
- Pomodoro Timer for individual students 
- Backend that allows students to log in and log hours spent doing assignments via the Pomodoro Timer

## How to Run/Use
1. Visit the website's link: *https://jhazmineaudrey.github.io/CSTT/*, no need to install anything on your local device.
2. Setup the timer in however way you wish.
3. Review upcoming classes via the dashboard on the right of the timer, or via the lower section.
